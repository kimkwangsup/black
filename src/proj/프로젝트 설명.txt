이 패키지는
오라클에 저장되어있는 게시판 관련 데이터를 조회, 수정, 삭제, 입력등의
작업을 처리할 프로젝트 패키지이다.

패키지 구조
	controller
		전체 작업을 컨트롤할 클래스가 정의되는 위치
			
			BoardController
			
	dao
		데이터베이스 작업을 처리할 클래스들이 정의되는 패키지
		
			BoardDao
	sql
		코드값을 알려주고 질의명령을 요청하면
		해당 코드값의 질의명령을 반환해주는 기능의 클래스 들로 
		구성된 패키지
			
			MemberSQL
			BoardSQL
			
	vo
		데이터 전달용 클래스 VO 클래스를 정의해 두는 패키지
		필요할 때 해당 클래스를 new 시켜서 데이터를 입력해서
		클래스째로 전달하면 된다.
			
			MemberVO
			BoardVO
